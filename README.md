# 🌤 Weather App - FR

Una aplicación web simple para consultar el clima actual de cualquier ciudad del mundo usando la API de OpenWeatherMap.

Desarrollada en **HTML, CSS y JavaScript puro**, permite a los usuarios buscar una ciudad y visualizar el estado del clima en tiempo real. La app muestra temperatura, humedad, velocidad del viento y un ícono representativo del clima actual. Además, el fondo cambia según el clima, y si se ingresa una ciudad que no existe, se muestra un mensaje de error claro y visible.
