# 🌤 Weather App - FR

Una aplicación web simple para consultar el clima actual de cualquier ciudad del mundo usando la API de OpenWeatherMap.

---

## 🧠 Descripción

**Weather App - FR** es una aplicación frontend desarrollada en **HTML, CSS y JavaScript puro**, que permite a los usuarios buscar una ciudad y visualizar el estado del clima en tiempo real. La app muestra temperatura, humedad, velocidad del viento y un ícono representativo del clima actual.

Además, el fondo cambia según el clima, y si se ingresa una ciudad que no existe, se muestra un mensaje de error claro y visible.

---

## 🚀 Funcionalidades

- 🔍 Buscar ciudad por nombre
- 🌡 Ver temperatura en °C
- 💧 Humedad del ambiente
- 🌬 Velocidad del viento
- 🎨 Fondo dinámico según clima
- ⚠️ Mensaje de error si no se encuentra la ciudad
- 🖱 Click en botón o tecla Enter

---
